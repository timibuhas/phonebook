# Agenda Telefonică – Proiect Atestat

Acesta este un proiect de tip **Agenda Telefonică** dezvoltat folosind **PHP pentru backend**, **MySQL pentru baza de date** și **React pentru frontend**. Proiectul permite gestionarea contactelor și a grupurilor, oferind funcționalități complete de adăugare, modificare, ștergere și căutare.  

Proiectul a fost realizat pentru **proiectul de atestat** din clasa a 12-a 

---

## Tehnologii utilizate
- **Backend:** PHP  
- **Frontend:** React.js  
- **Bază de date:** MySQL  

---

## Funcționalități principale
- Adăugarea, vizualizarea, editarea și ștergerea contactelor
- Crare de grupuri personalizate și adăugarea contactelor în acestea
- Căutare rapidă după nume sau număr de telefon  
- Interfață modernă și intuitivă realizată cu React  
- Persistența datelor în MySQL  


---

## Ce am învățat
- Crearea unei aplicații full-stack folosind **PHP + MySQL + React**  
- Conectarea frontend-ului React la API-uri PHP  
- Gestionarea și interogarea bazei de date MySQL  
- Organizarea proiectului în module și componente  
- Implementarea CRUD complet și validarea datelor
